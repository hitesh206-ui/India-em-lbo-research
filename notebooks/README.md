# Notebooks

This folder will hold the analysis notebooks for the project.

## Planned Notebooks

1. `01_market_overview.ipynb` - India and emerging-market private equity / buyout market overview.
2. `02_capital_structure_comparison.ipynb` - Comparison of India/EM and developed-market LBO capital structures.
3. `03_india_vs_us_lbo_model.ipynb` - Side-by-side standard US LBO model and India-adapted LBO model.
4. `04_exit_route_analysis.ipynb` - Exit-route comparison: IPO, strategic sale, secondary sale, and pending exits.
5. `05_regulatory_friction_score.ipynb` - Regulatory-friction score construction and sensitivity checks.

## Rule

Do not rely on unsupported assumptions in notebooks. Every external data point should be traceable to a source file, source URL, or documented assumption.
